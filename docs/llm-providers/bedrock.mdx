AWS Bedrock is not supported by LyraShield Engine. The optional upstream dependency and provider code are outside the LyraShield production contract. Use only an approved GPT-5.6 Sol, Terra, or Luna deployment through the documented OpenAI/Azure-compatible boundary.

Do not install upstream provider extras or forward AWS credentials to the engine worker.
