---
title: "Local models (unsupported upstream reference)"
description: "Not supported by LyraShield Engine"
---

<Warning>
Ollama, LM Studio, and other local model routes are not supported by LyraShield Engine. A locally hosted OpenAI-compatible API does not become supported unless its approved deployment is GPT-5.6 Sol, Terra, or Luna and it meets the controlled endpoint contract.
</Warning>

Do not treat local execution as equivalent result quality, privacy proof, or an air-gapped deployment claim.
